## Requisitos Funcionais (RF)

### RF01 — Cadastrar Cinema
O sistema deve permitir cadastrar unidades de cinema com nome, endereço, cidade, estado e capacidade.

### RF02 — Cadastrar Filme
O sistema deve permitir cadastrar filmes com título, duração, gênero, diretor e elenco.

### RF03 — Cadastrar Sessão
O sistema deve permitir registrar sessões de filmes em um cinema com horário e sala.

### RF04 — Registrar Público da Sessão
O sistema deve permitir registrar a quantidade de espectadores presentes em cada sessão.

### RF05 — Consultar Filmes em Cartaz
O sistema deve permitir visualizar os filmes exibidos em cada cinema.

### RF06 — Consultar Sessões de um Filme
O sistema deve permitir consultar todas as sessões de um determinado filme.

### RF07 — Consultar Público por Sessão
O sistema deve permitir visualizar o total de público de cada sessão.

### RF08 — Consultar Público por Filme
O sistema deve permitir visualizar o total de espectadores de um filme em todas as sessões.

### RF09 — Consultar Público por Cinema
O sistema deve permitir visualizar o total de público por unidade de cinema.
